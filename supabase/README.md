# Supabase local environment

Supabase is reserved for Auth, Postgres, private Storage, and local migrations. The CLI is not committed to the repository; run it through `npx supabase` and discover the installed command surface with `npx supabase --help` before initializing the local project.

The first database migration belongs to Step 2 (Auth, profiles, and RLS). Do not add production credentials or personal Huawei data to this folder.
